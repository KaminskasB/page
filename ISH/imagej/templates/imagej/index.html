About ImageJ Batch Processor

ImageJ Batch Processor is a macro for ImageJ that lets you select a directory and apply changes to all images found in that directory (and optionally all subdirectories).

By default, it will change brightness/contrast and apply the unmask sharpen filter. However, this is easily changed by inserting your commands in locations specified by the code comments. You can learn more about how to use the ImageJ macro language here (I recommend trying the command recorder).
