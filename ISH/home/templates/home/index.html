This page contains a couple of simple tools that automate some tasks I frequently need to do. Feel free to use them, but be aware they may not be properly tested for all input formats and edge cases yet and may give you weird results. Use at your own risk!

ImageJ Batch Processor is a macro for ImageJ that lets you select a directory and apply changes to all images found in that directory (and optionally all subdirectories).

The ISH probe melting temperature tool calculates RNA-RNA melting temperature during in situ hybridization.
